Social Media Content Generator

An automated workflow that uses AI to create LinkedIn posts about Software Testing in 2026, saves them to Google Sheets, and publishes to LinkedIn.

Download Workflow JSON View on GitHub Sample Google Sheet

What it does

1. AI picks a topicPlaywright, AI testing, QA trends, and more.
2. AI writes the postHook, body text, and image prompt.
3. Saves to Google SheetTitle, content, date, and status.
4. Posts to LinkedInVia Upload Post integration.

Quick setup

  1. Open n8n → Import → choose LinkedPostGeneration.json
  2. Add API keys for OpenRouter, Groq, Gemini, Google Sheets, and Upload Post
  3. Create a Google Sheet with columns: title | rationale | hook | status | Date | content | image_prompt
  4. Connect the sheet in the Google Sheets node
  5. Activate the workflow → open Chat → send a message to run

Important note

This site hosts the project documentation and workflow files. The automation itself runs in n8n (cloud or self-hosted), not on Vercel. Import the JSON file into your n8n account to use it.
